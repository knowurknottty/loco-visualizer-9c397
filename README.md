# 🐍 LOCO Visualizer

[![Deploy to Netlify](https://www.netlify.com/img/deploy/button.svg)](https://app.netlify.com/start/deploy?repository=https://github.com/knowurknottty/loco-visualizer)

A real-time visualization dashboard for **LOCO** — the Multi-Agent Swarm Intelligence system.

| **PCFE** | Predictive Consensus-Free Execution |
| **HDR** | Hyperspatial Distributed Reasoning |
| **TSCC** | Temporal-Spectral Consensus Computing |
| **NEUR** | Neuromorphic Unified Routing |
| **MFO** | Multi-Fidelity Optimization |

## 🎮 Two Viewing Experiences

### 🔷 Infinite Forge
- **Stack:** React 18 + Zustand + D3.js
- **Theme:** Dark Space (professional, clean)
- **Aesthetic:** Smooth animations, colorblind-safe palette
- **Target:** Architects, developers, researchers

### ⚡ Nexus Claw
- **Stack:** Vue 3 + Pinia + ECharts
- **Theme:** Cyberpunk (neon, glitch effects)
- **Aesthetic:** High contrast, wow factor
- **Target:** Visual enthusiasts, demo audiences

## 🚀 One-Click Deploy

Click the button above to deploy your own instance to Netlify instantly.

Or manually:

```bash
# Clone the repo
git clone https://github.com/knowurknottty/loco-visualizer.git
cd loco-visualizer

# Deploy to Netlify
netlify deploy --prod --dir=.
```
